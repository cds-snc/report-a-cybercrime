## Required Fields

You can mark required fields with a `required` prop. This will mark the field as required for the user. This prop does not add validation automatically, you still need to add validation against empty fields.

<Preview>
  <Story name="Required field">
    <Field
      name="name"
      label="This is a required field"
      helperText="Helper text"
      component={Input}
      required
    />
  </Story>
</Preview>
