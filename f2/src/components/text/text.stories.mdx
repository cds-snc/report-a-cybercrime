# Text

The text component can be used to wrap text when no wrapper is available

<Preview>
  <Story name="Text">
    <Text>Hello there</Text>
  </Story>
</Preview>

You have access to any [Chakra UI Text](https://chakra-ui.com/text) properties like `fontSize`

`<Text>` does not have spacing. This is useful when you don't want to use a `<P>` element. However, you can create spacing by making use of the [Chakra UI Stack](https://chakra-ui.com/stack) element

<Preview>
  <Story name="Font Size">
    <Stack spacing="4">
      <Text fontSize="sm">Hello there</Text>
      <Text fontSize="2xl">Hello there</Text>
      <Text fontSize="4xl">Hello there</Text>
    </Stack>
  </Story>
</Preview>
