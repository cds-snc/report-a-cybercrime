# Creating a new page

Pages are used to separate the user flow in small cognitive tasks. A page usually contains one thing, one action or one decision. A page usually leads to another page through buttons and links.

In this app, we create pages as exported components.

```jsx
export const Page = () => {
  return <Route render={({ history }) => <Page>Page Content</Page>} />
}
```

Every `<Page>` contains a mid-feedback form, rendered at the bottom of the page. Usually, you would want to add a `<Layout>` inside the page, then put your content inside the layout.

You should always have a H1 on your page. Other typical elements are a back button, a lead paragraph. You can then add a Form, more content, card containers, etc.
